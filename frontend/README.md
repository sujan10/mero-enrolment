# Mero Enrolment Frontend

This is the Next.js (App Router) frontend for the Mero Enrolment monorepo.

## Tech Stack
- Next.js (App Router)
- TailwindCSS
- shadcn/ui
- Zustand
- React Hook Form + Zod
- pdfjs-dist

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

The app will be available at http://localhost:3000
